Front-end development roadmap

In order to become a front-end developer one has to be aquinted with the following:-
  • How the internet works- look into domains, browsers, hosting and protocals
  • Tools Needed like the browsers to be used and how end users view the website
  • Editor majorly the VS Code Editor
  • Version Control , that is Git and GitHub
  1. HyperText Markup Language
  2. Cascading Styles Sheet
  3. Javascript
  4. Javascript Frameworks(React JS)

Front End development is wide and it should be user friendly and interactive.
